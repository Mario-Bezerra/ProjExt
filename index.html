Exercício 1: Contando com os Loops: Progressivo e Regressivo

Introdução Teórica: O Que São os Loops e Por Que Eles São Nossos Amigos Robôs!
Imagina que você tem um brinquedo bem legal, tipo um carrinho de controle remoto. E você quer que esse carrinho faça a mesma coisa várias e várias vezes, por exemplo, ir pra frente, virar, ir pra frente de novo e virar de novo. Você não ia querer ter que apertar os botões toda hora, né? Seria muito chato!
É aí que entram os loops! Pense nos loops como umas "instruções repetitivas" que a gente dá para o computador. É como se a gente dissesse: "Ei, computador! Faça isso aqui um montão de vezes sem parar, até eu te dizer para parar ou até você chegar em um certo número."
É tipo um robô muito obediente que faz uma tarefa repetida pra gente. Se você quer que ele diga "Olá!" 10 vezes, em vez de você ter que escrever "Olá!" 10 vezes, você só diz para o robô: "Robô, diga 'Olá!' 10 vezes!" E ele faz!
Neste exercício, a gente vai usar um tipo de loop que se chama for. O for é como se a gente dissesse para o robô: "Robô, PARA CADA uma dessas coisas (ou PARA CADA vez que eu te disser), faça isso!" Por exemplo: "PARA CADA dia da semana, tome café da manhã." Ou "PARA CADA amigo que vier na festa, dê um pedaço de bolo."
Então, os loops são nossos amigos robôs que nos ajudam a fazer tarefas chatas e repetitivas de um jeito super rápido e divertido! comuns.

Objetivo: Aprender a usar loops para contagens (progressiva e regressiva).

Resultado:

Instruções: Insira um número no campo 'Início' e outro no campo 'Fim', depois clique em 'Contar'.

Exemplo: Início: 1, Fim: 5 -> Saída: 1 2 3 4 5


        function contar() {
          let inicio = parseInt(document.getElementById("inicio1").value);
          let fim = parseInt(document.getElementById("fim1").value);
          let resultado = document.getElementById("resultado1");
        
          resultado.innerHTML = ""; // Limpa o resultado anterior
        
          if (isNaN(inicio) || isNaN(fim)) {
            resultado.innerHTML = "Por favor, insira números válidos.";
            return;
          }
        
          if (inicio < fim) {
            for (let i = inicio; i <= fim; i++) {
              resultado.innerHTML += i + " ";
            }
          } else if (inicio > fim) {
            for (let i = inicio; i >= fim; i--) {
              resultado.innerHTML += i + " ";
            }
          } else {
            resultado.innerHTML = "Os números são iguais!";
          }
        }

Exercício 2: Dominando a Tabuada com Loops

Introdução Teórica: Lembra que a gente falou que os loops são como robôs que fazem coisas repetitivas? Pois é! Agora a gente vai usar nosso amigo robô for para fazer algo super legal e que todo mundo já viu: a tabuada!
Imagina que você está aprendendo a tabuada do 2. Você sabe que é 2 vezes 1 igual a 2, 2 vezes 2 igual a 4, 2 vezes 3 igual a 6, e assim por diante. Mas e se a gente quisesse escrever a tabuada do 2 até o 100? Ia demorar uma eternidade pra gente escrever tudo, né?
Com o loop for, a gente pode pedir para o nosso robô fazer isso pra gente rapidinho! A gente pode dizer assim: "Robô, eu quero que você calcule a tabuada do 2. PARA CADA número de 1 até 10 (ou até onde eu quiser), multiplique o 2 por esse número e me diga o resultado!"
É como se o robô tivesse uma lista de números (1, 2, 3, 4...) e, PARA CADA número da lista, ele faz a continha de vezes. Ele é muito bom em matemática!
Então, os loops são incríveis não só para repetir coisas simples, mas também para fazer um montão de cálculos que seriam chatos e demorados pra gente fazer na mão! A tabuada é só um exemplo de como o nosso robô for pode nos ajudar a ser super rápidos com os números!

Objetivo: Gerar tabuadas automaticamente usando loops.

Tabuada:

    Instruções: Digite um número no campo 'Número' e clique em 'Gerar Tabuada'.

    Exemplo: Número: 7 -> Saída: 7 x 1 = 7, 7 x 2 = 14, etc.

    
function gerarTabuada() {
  let numero = parseInt(document.getElementById("numero2").value);
  let tabuada = document.getElementById("tabuada2");

  tabuada.innerHTML = ""; // Limpa a tabuada anterior

  if (isNaN(numero)) {
    tabuada.innerHTML = "Por favor, insira um número válido.";
    return;
  }

  for (let i = 1; i <= 10; i++) {
    let resultadoCalc = numero * i;
    let item = document.createElement("li");
    item.textContent = `${numero} x ${i} = ${resultadoCalc}`;
    tabuada.appendChild(item);
  }
}

    Exercício 3: Somando Números com Loops: Calculando Totais

    Introdução Teórica: Neste exercício final, usaremos o loop for para realizar uma soma acumulativa, somando vários números em sequência dentro de um intervalo.
    O que é isso? Imagina que você tem uma caixinha de tesouros vazia. Aí você encontra uma moeda de 1 real e coloca na caixinha. Agora você tem 1 real. Depois, você encontra mais uma moeda de 2 reais e coloca na caixinha. Agora você tem 1 real mais 2 reais, que dá 3 reais. Depois mais uma moeda de 3 reais, e assim por diante. Você está acumulando (juntando) o valor dos tesouros na sua caixinha.
    Com o loop for, a gente pode fazer o computador fazer essa mesma coisa! A gente pode dizer para ele: "Computador, comece com um zero na sua caixinha. Agora, PARA CADA número que eu te der (por exemplo, os números de 1 até 5), adicione esse número na sua caixinha!"

    Objetivo: Calcular a soma de números em um intervalo usando loops.

    Resultado da Soma:

    Instruções: Digite o início e o fim do intervalo, depois clique em 'Somar'.

    Exemplo: Início: 1, Fim: 10 -> Saída: A soma do intervalo é: 55

    
        function somarIntervalo() {
          let inicio = parseInt(document.getElementById("inicio3").value);
          let fim = parseInt(document.getElementById("fim3").value);
          let resultado = document.getElementById("resultado3");
        
          if (isNaN(inicio) || isNaN(fim)) {
            resultado.textContent = "Por favor, insira números válidos.";
            return;
          }
        
          let soma = 0;
          let menor = Math.min(inicio, fim);
          let maior = Math.max(inicio, fim);
        
          for (let i = menor; i <= maior; i++) {
            soma += i;
          }
        
          resultado.textContent = `A soma do intervalo é: ${soma}`;
        }
